# Games-Awards-Goty-Nominees-and-Winners-2014-2024

The dataset includes key information for each nominated game, including:

- Name: The title of the game.
- Developer and Publisher: The companies behind the game.
- Genre: The primary genre (Action-Adventure, RPG, etc.).
- Release-Date: The exact release date.
- Year of the Event: The year of The Game Awards ceremony in which the game was nominated (e.g., 2020).
- Meta-Score: The aggregated critic score from Metacritic (0-100).
- Reviews: The total number of critic reviews used for the Metascore.
- User-Score: The aggregated user score from Metacritic (0-100).
- Votes: The total number of user ratings/analyses used for the User-Score.
- Popularity: A metric indicating relative search interest, obtained from Google Trends.
- Most-Anticipated: Indicates if the game was nominated for "Most Anticipated Game".
- Nominations: The total number of nominations the game received at that event.
- Wins: Whether the game won (1) or lost (0) the main GOTY award.
